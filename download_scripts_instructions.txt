## Instrucciones de Instalación

1. **Descarga el Archivo ZIP** completo con todos los scripts.

2. **Descomprime el Archivo** en una carpeta en tu computadora.

3. **En Second Life**:
   - Crea objetos para cada componente (HUD, Meter, NPCs, objetos interactivos)
   - Copia y pega el script correspondiente en cada objeto
   - Configura los permisos según las necesidades de tu roleplay

4. **Instalación por Componentes**:
   - **HUDs**: Deberán ser configurados como attachments personales
   - **Meters**: Deben configurarse como objetos flotantes sobre la cabeza
   - **Objetos Interactivos**: Colócalos en las ubicaciones apropiadas del sim
   - **NPCs**: Configúralos como avatares con los scripts correspondientes

5. **Configuración de Canales**:
   - Asegúrate de que todos los componentes utilicen el mismo canal base
   - El valor predeterminado es -98765, pero puedes cambiarlo
   - Este cambio debe hacerse en TODOS los scripts para que funcionen juntos

## Notas Importantes

- **Permisos LSL**: Verifica que todos los objetos tengan los permisos necesarios
- **Memoria**: Monitorea el uso de memoria de los scripts más complejos
- **Personalización**: Modifica los nombres y títulos según las necesidades de tu roleplay
- **UUIDs Específicos**: Algunos scripts tienen UUIDs codificados para roles especiales (Tsar, Tsarevich)
- **Respaldos**: Mantén copias de respaldo de los scripts antes de hacer modificaciones
